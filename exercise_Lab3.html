Answer the following questions

1. What style would you use to change the font of a block quote to Times Roman or if Times Roman not available. to change to serif font?

font-family: "Times New Roman", Times, serif;



2. What style would you use to set the size of paragraph text to 12 point type?

<p style="font-size:12pt;">paragraph</p>"



3. What style would you use to underline all your headings and display heading text in uppercase letter?

<h1 style="text-decoration: underline;text-transform: uppercase;">heading</h1>



4. What style would you use to center your paragraph text and display it in boldface italics?

<p style="text-align:center;"><Strong><i> Any text here! </i> </Strong></p>
